# Master_Ivanti_AzureSnapshotAndPatch.ps1
Master Ivanti script that takes Azure snapshots, and kicks off Ivanti job subscripts. All 14 of the secondary scripts I use in my example are identical as far as the script is concerned. Each script correlates to 1 Ivanti Machine Group.

**DNS** is used for Domain Naming System as an abbreviation for domain identifier. This script has nothing to do with DNS in a networking sense.
